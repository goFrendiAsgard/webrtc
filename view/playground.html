Before starting, please:

  • You are on google chrome
  • Make sure you have arduino leonardo connected to your computer
  • Make sure pin 12 is connected to push button, and pin 13 is connected to LED
  • Open chrome://flags in other tab, and enable Experimental Web Platform Features (you might need to restart chrome)
  • Open chrome://device-log in other tab
  • Open developer tools (ctl + shift + i)
  • Click start button
  • Choose arduino leonardo
  • If you are using linux, and you got Access dennied error, open terminal and perform chmod 777 /dev/bus/usb/001/007, you can get the device address by looking at device-log
  • If you are using windows, good luck :)
